# Student Social Media Addiction Analysis Dashboard

## Project Overview

This project analyzes the impact of social media usage on students' mental health, sleep habits, relationships, and academic performance using an interactive Power BI dashboard. The objective was to identify behavioral patterns associated with excessive social media use and present actionable insights through data visualization.

## Dataset

* Total Records: 705 Students
* Data Sources: Student Details and Platform Details
* Key Attributes: Age, Gender, Academic Level, Daily Usage Hours, Addiction Score, Mental Health Score, Sleep Hours, Most Used Platform, Relationship Status, Academic Impact, and Social Media Conflicts.

## Tools & Technologies

* Power BI
* DAX (Data Analysis Expressions)
* Data Modeling
* Data Visualization
* Microsoft Excel

## Dashboard Features

* Executive Overview with KPI Cards
* Mental Health & Lifestyle Analysis
* Academic Performance Impact Assessment
* Relationship & Social Conflict Analysis
* Interactive Slicers, Bookmarks, and Drill-through Pages
* Platform-wise Usage and Behavioral Insights

## Key Insights

* 64.26% of students reported that social media usage negatively affected their academic performance.
* Students spend an average of 4.92 hours per day on social media.
* Average sleep duration was 6.87 hours per night.
* Higher addiction scores showed a negative relationship with mental health scores.
* Platform preferences varied across demographic groups, influencing usage patterns and conflict levels.
